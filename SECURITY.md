# Security Policy

## Supported Versions
AuroraStack follows semantic versioning. Security fixes are provided for the latest minor release line.

## Reporting a Vulnerability
Please report security issues **privately**.

- Email: security@cajeer.com
- Subject: [AuroraStack] Security report

Include:
- A clear description of the issue
- Steps to reproduce (PoC if possible)
- Affected versions and components
- Impact assessment (what can be achieved)

We aim to acknowledge reports within 72 hours.

## Coordinated Disclosure
We will coordinate a fix and publication timeline with the reporter when feasible.

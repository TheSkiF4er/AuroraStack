# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-13
### Added
- Initial public release of the AuroraStack monorepo scaffold.
- Core package boundaries and public entrypoints for framework modules.
- Playground app and basic template to validate end-to-end wiring.
- Release engineering: pnpm workspaces, Turborepo, Changesets, CI workflows.

### Notes
- AuroraStack is designed to evolve via packages and plugins while keeping the core small.
